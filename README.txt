Proyecto:
Visualización, Análisis y Modelo Predictivo de Datos de las Declaraciones de Importación y Exportación de Colombia, a través de la creación de una Base de Datos con MongoDB donde se almacenan todos los registros suministrados por la DIAN (Aplicativo para la Conversión de CSV a JSON e Inserción de los Documentos en las Colecciones de la Base de Datos), además de la creación de un Dashboard Full Stack con MERN en donde el Backend fue realizado usando Node JS, Express JS y Mongoose; y para la creación del FrontEnd se utilizó React, Material UI, Nivo Charts, Redux Toolkit, Material UI Data Grid y Redux Toolkit Query.
